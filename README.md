# 🌸 Học Tiếng Trung Cùng Nana (HSK 1)

Ứng dụng học tiếng Trung miễn phí dành cho người Việt — **Learn Chinese with Nana**.

## ✨ Tính năng

- 📚 **Từ vựng HSK 1** — Bài 1, Bài 2, Bài 3 (preview)
- 🎯 **Trắc nghiệm** — Dịch Trung ↔ Việt
- ✍️ **Gõ chữ** — Tự gõ câu trả lời
- 🎧 **Nghe chính tả** — Giọng nữ tiếng Trung đại lục (zh-CN)
- 🌸 Giao diện cute, hồng pastel

## 🚀 Cách dùng

Mở file `index.html` trên trình duyệt **Chrome** hoặc **Edge** (để hỗ trợ giọng đọc TTS tốt nhất).

Hoặc xem online tại: https://yinadu13-del.github.io/hsk1

## 📖 Nội dung bài học

| Bài | Chủ đề | Số từ |
|-----|--------|-------|
| Bài 1 | Xin chào! 你好 | 4 từ |
| Bài 2 | Xin lỗi & Không sao 对不起 / 没关系 | 12 từ |
| Bài 3 | Bạn khỏe không? 你好吗 | 34 từ |

## 💡 Lưu ý

- Dùng **Chrome hoặc Edge** để nghe TTS tốt nhất
- 他/她/它 đọc giống nhau (tā) — nghe chính tả chấp nhận cả 3 chữ
